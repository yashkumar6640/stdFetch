Request Basics

  
  A request can be initiated by invoking the appropriate method on stdFetch object, like
  
  stdFetch.get() 
  stdFetch.post 
  stdFetch.put()
  stdFetch.patch()

  

  then calling .then() to process the data received in response. You can work with data such as

   json()
   blob()
   formData()
   text()


  Basic Example:
  
    stdFetch.get("https://jsonplaceholder.typicode.com/posts/1")
    .then(response => {
      // res.body, res.headers, res.status
      console.log(response);
    })
